# 11.1 命令系统架构与扩展

## 命令系统概述

Claude Code 的命令系统为常见工程任务提供预制工作流。命令是高级抽象，利用工具系统实现具体功能。

## 1. 命令接口定义

**文件**: `src/commands.ts:1-300`

```typescript
/**
 * 命令接口
 */
export type Command =
  | PromptCommand
  | ToolCommand

/**
 * 基于提示的命令（通过对话驱动）
 */
export interface PromptCommand {
  type: 'prompt'
  name: string
  shortName?: string              // 简短别名
  description: string
  progressMessage: string
  aliases?: string[]              // 别名列表

  // 可选：限制可用工具
  allowedTools?: string[]

  /**
   * 生成用于对话的提示
   */
  getPromptForCommand(
    args: string,
    context: CommandContext
  ): Promise<ContentBlock[]>
}

/**
 * 基于工具的命令（直接执行）
 */
export interface ToolCommand {
  type: 'tool'
  name: string
  description: string

  /**
   * 直接执行命令
   */
  invoke(args: string, context: CommandContext): Promise<void>
}

/**
 * 命令执行上下文
 */
export interface CommandContext {
  appState: AppState
  setAppState: SetAppState
  tools: Tool[]
  commands: Command[]
  canUseTool: CanUseToolFn
  abortController: AbortController
  executeCommand: (name: string, args: string) => Promise<string>
}
```

### 2.1 /commit 命令

**文件**: `src/commands/commit/commit.ts:1-400`

```typescript
export const CommitCommand: PromptCommand = {
  type: 'prompt',
  name: 'commit',
  shortName: 'c',
  description: 'Create a git commit',
  progressMessage: 'creating commit',
  aliases: ['git-commit', 'git commit'],

  allowedTools: [
    'bash',           // 用于 git 命令
    'read',           // 查看 diff
    'glob'            // 查找文件
  ],

  async getPromptForCommand(args, context) {
    // === 收集 Git 上下文 ===
    const [gitStatus, gitDiff, recentCommits] = await Promise.all([
      executeCommand('git status --short'),
      executeCommand('git diff --cached', context.appState.cwd),
      executeCommand('git log --oneline -10')
    ])

    // === 构建提示 ===
    const prompt = `
## Git Context

Current Status:
\`\`\`
${gitStatus}
\`\`\`

Recent Commits:
\`\`\`
${recentCommits}
\`\`\`

Staged Changes:
\`\`\`
${gitDiff}
\`\`\`

## Task

Create a single git commit:
1. Analyze the staged changes
2. Follow the repository's commit style
3. Create a concise commit message (1-2 sentences)
4. Execute: git commit -m "message"

Rules:
- Do NOT use --amend unless explicitly requested
- Do NOT skip hooks (--no-verify)
- Do NOT commit secrets (.env, credentials.json)
${args ? `\nAdditional context: ${args}` : ''}
    `.trim()

    return [{ type: 'text', text: prompt }]
  }
}
```

### 2.3 /mcp 命令

```typescript
export const McpCommand: ToolCommand = {
  type: 'tool',
  name: 'mcp',
  description: 'Manage MCP servers',

  async invoke(args, context) {
    const subcommand = args.split(' ')[0]

    switch (subcommand) {
      case 'list':
        // 列出所有已配置的 MCP 服务器
        const configs = await loadMCPConfigs()
        console.log('Configured MCP servers:')
        Object.entries(configs).forEach(([id, config]) => {
          console.log(`  - ${id}: ${config.name}`)
        })
        break

      case 'connect':
        // 连接到 MCP 服务器
        const serverId = args.split(' ')[1]
        const client = new MCPClient()
        await client.ensureConnected(serverId)
        console.log(`Connected to ${serverId}`)
        break

      case 'test':
        // 测试 MCP 服务器连接
        const testId = args.split(' ')[1]
        const testClient = new MCPClient()
        const connection = await testClient.ensureConnected(testId)
        console.log(`Server capabilities: ${JSON.stringify(connection.capabilities)}`)
        break

      default:
        console.log('Unknown MCP subcommand')
    }
  }
}
```

## 3. 命令注册和执行

**文件**: `src/commands.ts:300-500`

```typescript
/**
 * 所有内置命令
 */
const BUILTIN_COMMANDS: Record<string, Command> = {
  // === 代码操作 ===
  'commit': CommitCommand,
  'review': ReviewCommand,
  'commit-push-pr': CommitPushPrCommand,

  // === MCP 管理 ===
  'mcp': McpCommand,

  // === 项目设置 ===
  'init': InitCommand,

  // === 任务管理 ===
  'tasks': TasksCommand,

  // === 代理管理 ===
  'agents': AgentsCommand,

  // === 内存管理 ===
  'memory': MemoryCommand,
  'context': ContextCommand,

  // === 诊断 ===
  'doctor': DoctorCommand,
  'debug': DebugCommand,

  // ... 更多命令
}

/**
 * 执行命令
 */
export async function executeCommand(
  commandName: string,
  args: string,
  context: CommandContext
): Promise<void> {
  // === 查找命令 ===
  const command = Object.values(BUILTIN_COMMANDS).find(c =>
    c.name === commandName ||
    (c.type === 'prompt' && (c as PromptCommand).aliases?.includes(commandName)) ||
    (c.type === 'prompt' && (c as PromptCommand).shortName === commandName)
  )

  if (!command) {
    throw new Error(`Command not found: ${commandName}`)
  }

  // === 检查权限 ===
  // 某些命令可能需要特殊权限

  // === 执行命令 ===
  if (command.type === 'prompt') {
    // 基于提示的命令：通过对话执行
    const promptCommand = command as PromptCommand

    // 获取提示内容
    const promptContent = await promptCommand.getPromptForCommand(args, context)

    // 使用 QueryEngine 执行对话
    await queryClaudeWithTools(
      promptContent[0].text || '',
      '',
      // 如果命令限制了工具，使用限制工具列表
      promptCommand.allowedTools
        ? context.tools.filter(t => promptCommand.allowedTools!.includes(t.name))
        : context.tools,
      {
        appState: context.appState,
        abortController: context.abortController
      }
    )
  } else {
    // 工具命令：直接执行
    const toolCommand = command as ToolCommand
    await toolCommand.invoke(args, context)
  }
}

/**
 * 解析斜杠命令
 */
export function parseCommand(input: string): { name: string; args: string } | null {
  if (!input.startsWith('/')) {
    return null
  }

  const parts = input.slice(1).split(' ', 1)
  const name = parts[0]
  const args = input.slice(name.length + 2) // +2 for '/' and space

  return { name, args }
}
```

## 4. 命令扩展

```typescript
/**
 * 自定义命令示例

export const CustomCommand: PromptCommand = {
  type: 'prompt',
  name: 'custom',
  description: 'My custom command',
  progressMessage: 'running custom command',

  async getPromptForCommand(args, context) {
    return [{
      type: 'text',
      text: `Custom task: ${args}`
    }]
  }
}

/**
 * 注册自定义命令
 */
export function registerCustomCommand(command: Command): void {
  BUILTIN_COMMANDS[command.name] = command
}
```

## 关键特性

✅ **两种类型** - 提示命令和工具命令
✅ **灵活执行** - 支持别名和简称
✅ **工具限制** - 命令可限制可用工具
✅ **可扩展** - 易于添加自定义命令
✅ **高级抽象** - 隐藏复杂工具逻辑
